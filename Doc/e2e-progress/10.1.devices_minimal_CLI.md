=== 1. Minimal CLI code
```shell
# 1) Tạo device
curl -s -X POST http://localhost:8080/devices `
  -H "Content-Type: application/json" `
  -d '{ "name": "dev-1", "type": "sensor", "metadata": {"site":"A"} }'

# 2) Lấy device theo id (ví dụ id=1)
curl -s http://localhost:8080/devices/1

# 3) Thêm reading cho device
curl -s -X POST http://localhost:8080/readings `
  -H "Content-Type: application/json" `
  -d '{ "device_id": 1, "value": 42.5, "ts": "2025-08-25T10:00:00Z" }'

# 4) Lấy last reading
curl -s http://localhost:8080/devices/1/readings/last

# 5) Tính alerts (compute)
curl -s -X POST http://localhost:8080/alerts/compute

# 6) Xem danh sách alerts
curl -s http://localhost:8080/alerts

# 7) Đánh dấu serviced (ví dụ alert id=1)
curl -s -X POST http://localhost:8080/alerts/1/serviced
```
